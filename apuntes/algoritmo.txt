    -   El servidor debe lanzarse primero y mostar su PID

    - Recibe como parametros el PID y el string. Este string ha de enviarse al servidor

/*  SERVER  */
    - Muestra el string recibido
    - Envia ACK en caso de que se haya recibido

	ARQ / S&W

	SIGURS2 -> bit 0 (down)
